# テスト仕様書 — calculator.ts

## 1. 概要

| 項目 | 内容 |
|------|------|
| ドキュメントID | TS-001 |
| 対象モジュール | `src/utils/calculator.ts` |
| 作成日 | 2026-03-11 |

## 2. テスト目的

`calculator.ts` に定義された各関数が、正常系・異常系・境界値において仕様通りに動作することを検証する。

## 3. テスト対象範囲

### 対象
| 関数名 | 概要 |
|--------|------|
| `add(a, b)` | 2つの数値の加算 |
| `subtract(a, b)` | 2つの数値の減算 |
| `multiply(a, b)` | 2つの数値の乗算 |
| `divide(a, b)` | 2つの数値の除算（0除算エラーあり） |
| `isPositiveInteger(value)` | 正の整数判定 |
| `sum(numbers)` | 配列の合計（空配列エラーあり） |
| `average(numbers)` | 配列の平均（空配列エラーあり） |

### 対象外
- UIコンポーネント（`Counter.vue`）のテスト → 結合テスト仕様書にて対応
- Composable（`useCounter.ts`） → 別の単体テスト仕様書にて対応

## 4. テスト手法

| 手法 | 適用範囲 |
|------|---------|
| 単体テスト | 各関数を個別に検証 |
| 同値分割 | 入力を有効/無効クラスに分類 |
| 境界値分析 | 0, 負の数, 小数点などの境界付近 |
| エラー推測 | 0除算, 空配列 |

## 5. テスト環境

| 項目 | 内容 |
|------|------|
| テストフレームワーク | Vitest |
| テストファイル | `tests/unit/calculator.spec.ts` |
| 実行コマンド | `yarn test tests/unit/calculator.spec.ts` |

## 6. 合格基準

- [x] 全テストケースがパスすること
- [x] 各関数の正常系・異常系・境界値がカバーされていること
- [x] エラーメッセージが仕様通りであること
